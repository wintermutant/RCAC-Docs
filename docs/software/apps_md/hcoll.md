# hcoll

[Back to application catalog](/software/app_catalog/)

## Description

Modern interface for Mellanox Fabric Collective Accelerator (FCA). FCA is a MPI-integrated software package that utilizes CORE-Direct technology for implementing the MPI collective communications.

## Homepage

[https://developer.nvidia.com/networking/hpc-x](https://developer.nvidia.com/networking/hpc-x){:target="_blank"}

## Module

You can load the module by:

```bash
module load hcoll
```

!!! note "Note for using `hcoll`"
    Run `module spider hcoll` beforehand to check if this version requires any prerequisite modules.
